# Machines

Everything needed to build a CocktailBerry machine: 3D-printable parts (STLs), preparation steps, and assembly guides.
Each machine has its own section, including the parts needed for that build.

## Available Machines

- [CocktailBerry MK III](mk3/index.md): 3D-printed with a compact footprint and integrated touchscreen support.
- [CocktailBerry MK IV](mk4/index.md): The latest v2-focused reference build with a cleaner design and optional scale plus peristaltic pump support.
- [CocktailBerry 2-Go](2-go/index.md): A compact portable build in a Euro Box with detachable legs optimized for transport.
