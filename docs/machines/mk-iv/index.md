# CocktailBerry MK IV

The **MK IV** is the upcoming CocktailBerry reference build, designed around the
CocktailBerry **v2** software. Like the MK III, it moves the wiring onto the
[CocktailBerryBoard](../../pcbas/cocktailberry-board.md).

_Documentation is a work in progress._

--8<-- "machine/alpha.md"

## Specifications

| Property   | Value                                          |
| ---------- | ---------------------------------------------- |
| Dispensers | 8 × membrane pumps 2x peristaltic pumps        |
| Display    | Tablet or Smart Phone                          |
| Controller | Raspberry Pi 4 (newer models also work)        |
| Power      | 12 V input; internal transformer powers the Pi |
| Software   | CocktailBerry v2                               |
| Enclosure  | Fully 3D-printed                               |
| Dimensions | _TBD_                                          |
