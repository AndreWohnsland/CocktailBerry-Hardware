# Preparation

Steps to complete before assembly.

## Printing

- Recommended material: _e.g. PETG_
- Layer height: _e.g. 0.2 mm_
- Infill: _e.g. 20 %_
- Supports: _where needed_

## Post-Processing

Sanding, drilling, threading inserts, etc.

## Pre-Assembly Checks

- [ ] All parts printed and cleaned
- [ ] Electronics tested
- [ ] Fasteners and tools ready
