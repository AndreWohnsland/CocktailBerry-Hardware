# Accessories

Additional hardware to extend a CocktailBerry machine - add-ons that are not tied to one specific machine build.
This section is just getting started; accessories will be added over time.

## Setup

Setup instructions for each accessory will be added here as they are released.

## Downloads

The printable and CAD files are attached to each [release]({{extra.repo_url}}/releases) as a single archive:

- **3D files:**
  [`accessories.zip`]({{extra.repo_url}}/releases/latest/download/accessories.zip)
  - STL (print-ready) and STEP (CAD) for every part.
